# AI浏览器

一个功能丰富的安卓浏览器应用程序，支持多种搜索引擎、语音识别、文字朗读、浏览器标识选择、手势控制等功能。该应用能够识别并直接打开用户输入的网址，实现了搜索和浏览的一体化功能。

## 主要功能特性

### 1. 搜索与浏览一体化
- 支持9种搜索引擎：百度、谷歌、必应、秘塔AI、夸克、搜狗、360、头条、夸克AI
- 智能URL识别：当用户输入网址时，应用会自动识别并直接打开网页而非执行搜索
- 搜索框支持回车搜索功能

### 2. 语音功能
- 语音输入搜索内容
- 文字朗读：支持网页内容朗读功能
- 语音识别API支持在线APIJ1UoskﻺAPK
### 3. 浏览器标识选择
- 可在设置中选择不同的浏览器标识：Android（手机）、Android(平板)、Windows (Chrome)、Windows (IE 11)、macOS、iPhone、iPad、塞班(Symbian)、自定义

### 4. 手势控制
- 上下滑动手势：控制顶部按钮显示/隐藏
- 左右滑动手势：支持前进/后退功能

### 5. UI设计
- 顶部按钮栏采用黑色背景，白色图标
- 包含后退、前进、刷新、搜索框、搜索按钮、搜索引擎按钮之间有适当的间距
- 状态栏始终隐藏

## 构建说明

由于在容器环境中构建Android项目受限（AAPT2 daemon无法启动等问题），建议采用云端构建方案：

1. 将项目推送至GitHub仓库
2. 确保.gitignore正确配置，不忽略关键构建文件
3. 配置GitHub Actions工作流实现云端构建
4. 在云端环境中使用最新版本的cmdline-tools避免404错误
